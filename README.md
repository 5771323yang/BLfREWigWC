# 项目简介

## 核心功能
- 基于YOLOv8的焊缝质量检测，支持识别6类缺陷：['Bad Welding', 'Crack', 'Excess Reinforcement', 'Good Welding', 'Porosity', 'Spatters']
- 支持3种检测方式：单张图片、图片文件夹、视频文件、本地摄像头
- 支持自定义检测参数：置信度、IOU阈值
- 提供UI界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 可选择不同检测模型

## 适用场景
- 焊接质量检测与评估
- 工业自动化质检流程
- 焊缝缺陷研究与分析
- 制造业质量控制

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于python3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 内置1300张焊缝缺陷数据集
- 提供完整源码、训练好的模型及1300张标注数据集
- 包含UI界面源文件、图标、项目运行教程等完整文件
- 已配置成功环境，可直接使用
